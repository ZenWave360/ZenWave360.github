# ZenWave Editor - IntelliJ Plugin

The ZenWave Editor IntelliJ Plugin is a plugin for IntelliJ IDEA that allows you to edit ZenWave files with syntax highlighting and code completion.

With the ability to run ZenWave SDK directly from the editor and export to markdown, including PlantUML diagrams.
